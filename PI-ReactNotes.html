Welcome to, React Notes

a simple jotting tool

React Notes was created to help users quickly write notes. React Notes is a virtual sticky note app. Make it clear to yourself what your tasks are each day, and add as many notes as you need.

React Notes lets the user create a title for the specific note and a description. Once the user has completed their inputs the plus button will add it to the list of all notes below the input form. Once a user has completed a tasks or no longer needs the note, the user can then delete the note by clicking on the red trash can icon.

React Notes was made using HTML, CSS, and React. React Notes can be very useful, but the user must remember this is a stateless application so none of the information will be retained after the page is refreshed.
